### ✅ 1. 文案库编写（145+条温暖文案）

**文档：** `文案库_温暖提醒文案.md`

**文案分类（8大类）：**

```
1. 记录完成文案（15+条）
   ├─ 首次记录鼓励
   ├─ 连续记录激励
   └─ 动机记录完成

2. 时长提醒文案（60+条）
   ├─ 30分钟：温柔陪伴（5+条）
   ├─ 60分钟：友善建议（10+条）
   ├─ 90分钟：温和监督（15+条）
   ├─ 120分钟：明确监督（20+条）
   └─ 180分钟+：强监督（10+条）

3. 能量等级文案（20+条）
   ├─ 90-100：精力充沛
   ├─ 70-89：状态不错
   ├─ 50-69：还行吧
   ├─ 30-49：有点累了
   └─ 0-29：快撑不住了

4. 晚报文案（30+条）
   ├─ 开场文案（基于能量）
   ├─ "我说了算"占比文案
   ├─ 动机分析文案
   └─ 总结文案

5. 引导行动文案（10+条）
   ├─ 能量恢复建议
   └─ 微任务建议

6. 用户交互回复文案（10+条）

7. 特殊场景文案（10+条）
   ├─ 首次使用欢迎
   ├─ 连续未使用
   ├─ 连续使用7天
   └─ 特殊成就

8. 文案使用指南
   ├─ 文案选择逻辑
   └─ 文案组合规则
```

**核心原则：**
```
1. 永远不批评
   ❌ "你刷太多了"
   ✅ "刷了X小时了，累不累？"

2. 具体行动代替抽象建议
   ❌ "去做点有意义的事"
   ✅ "站起来倒杯水"

3. 温柔而明确
   ❌ "你必须停下来！"
   ✅ "说实话，你需要休息了❤️"

4. 给用户选择权（初期）
   30-90分钟：可以忽略
   90分钟+：逐渐收紧选择

5. 监督时保持温度
   即使是强提醒，也带❤️
```

**文案示例：**

```
30分钟（温柔）：
"刷了30分钟了，累不累？
要不要：
• 起来倒杯水 💧
• 看看窗外 🌳
• 伸个懒腰 🙆"

90分钟（温和监督）：
"🌳 刷了1.5小时了
眼睛累不累？大脑是不是有点晕？
试试这个：
1. 站起来，走到窗边
2. 看看最远的地方（至少20秒）
3. 深呼吸3次"

180分钟+（强监督）：
"🚨 刷了3小时了
这不是建议，是提醒：
你需要休息了。
现在：
1. 站起来
2. 离开手机
3. 出去走走（至少5分钟）"
```

### ✅ 2. 界面原型设计文档

**文档：** `界面原型设计文档.md`

**核心页面设计（5个）：**

#### **1. 首页（Home）**
```
布局：
├─ 顶部：标题
├─ 能量卡片（占1/3屏幕）
│   ├─ 能量进度条（65/100）
│   ├─ 能量等级（😊 状态不错）
│   └─ 实时动画更新
├─ 今日统计
│   ├─ 刷短视频次数/时长（-能量）
│   └─ 专注做事次数/时长（+能量）
├─ 主要操作按钮
│   ├─ [记录刷短视频]（主按钮）
│   └─ [记录做事]（次按钮）
├─ "我说了算"占比卡片
│   ├─ 圆形进度条（60%）
│   └─ 趋势对比（↑5%）
└─ 底部导航
    └─ [首页] [晚报] [我的]
```

#### **2. 记录弹窗**
```
表单：
├─ 平台选择（抖音/视频号/小红书/B站）
├─ 时长选择（15/30/60/90分钟或自定义）
├─ 动机选择（😌累了/😐无聊/😔逃避/📚学习）
├─ 类型选择（◉我说了算 / ○被带着走）
├─ 预计影响提示（能量 65→50 -15）
└─ [取消] [确认记录]

特点：
├─ 快捷选择（3秒完成）
├─ 实时计算能量影响
└─ 记住上次选择
```

#### **3. 提醒弹窗（三种样式）**
```
样式A：温柔提醒（30-60分钟）
├─ 小弹窗（占屏幕1/3）
├─ 温柔文案
└─ 双选择按钮

样式B：温和监督（90-120分钟）
├─ 中等弹窗（占屏幕1/2）
├─ 带图标提示
└─ 具体行动步骤

样式C：强监督（180分钟+）
├─ 全屏弹窗
├─ 警示色
└─ 单选择按钮（必须回应）
```

#### **4. 晚报页面**
```
内容：
├─ 标题（🌙 今日能量报告）
├─ 能量变化曲线（折线图）
├─ 最终能量状态
├─ 今日统计（刷短视频/专注做事）
├─ "我说了算"占比（圆形进度+趋势）
├─ 动机分析
│   ├─ 主要动机识别
│   ├─ 洞察文案（引发思考）
│   └─ 引导性问题
└─ 明天目标设定
```

#### **5. 设置页面**
```
功能：
├─ 提醒方式调整（陪伴-监督比例滑动条）
├─ 提醒时机自定义（30/60/90分钟）
├─ 外观设置（浅色/深色/自动）
├─ 数据管理（导出/清除）
└─ 关于（版本/GitHub/隐私政策）
```
