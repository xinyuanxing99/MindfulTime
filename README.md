### 💬 一句话介绍

> **"刷了2小时了，累不累？起来倒杯水吧"** —— 朋友式陪伴，而非冷酷监控

### 🎯 为什么做这个项目？

```
发现：刷短视频停不下来，不是自制力问题
而是：你在逃避什么？累了？无聊？压力太大？

传统工具：批评你"刷太久了！"
我们：温柔询问"你在逃避什么？"

让改变，从觉察开始 🌱
```

### 📱 核心功能

#### 1️⃣ 记录动机（帮你看见真相）

```
选择平台：抖音 / 视频号 / 小红书 / B站
输入时长：15 / 30 / 60 / 90分钟
选择动机：
  • 累了想放松 😌
  • 无聊打发时间 😐
  • 逃避工作/学习 😔
  • 主动学习 📖

预计影响：能量 65 → 50 (-15)
```

#### 2️⃣ 能量条（可视化精力）

| 能量等级 | 分数 | 状态 |
|---------|------|-----|
| 🌟 精力充沛 | 90-100 | 今天状态真好！ |
| 😊 状态不错 | 70-89 | 继续保持~ |
| 😐 还行吧 | 50-69 | 要不要休息下？ |
| 😔 有点累了 | 30-49 | 你应该很累了吧？ |
| 💤 快撑不住了 | 0-29 | 你真的需要休息了 |

#### 3️⃣ 三阶梯提醒（渐进式引导）

```
⏱️ 30分钟  →  温柔询问
   "刷了30分钟了，累不累？"
   [我去休息] [再刷一会儿]

⏱️ 60分钟  →  友善建议
   "刷了1小时了，试试这些？"
   • 起来倒杯水💧
   • 看看窗外🌳
   [我去做] [我知道了]

⏱️ 90分钟  →  温和监督
   "⚠️ 刷了1.5小时了
   认真问你：记得刚才看了什么吗？"
   [我现在去] [再给我10分钟]

⏱️ 180分钟  →  必须休息
   "🚨 刷了3小时了
   说实话，你需要休息了 ❤️"
   [我现在去]（只有1个选择）
```

## 🌟 设计哲学

### 1. 温柔而明确

```
❌ "你已经刷了2小时了，太多了！"
✅ "刷了2小时了，累不累？起来走走吧？"

❌ "你必须停下来！"
✅ "说实话，你需要休息了 ❤️"
```

### 2. 具体行动代替抽象建议

```
❌ "去做点有意义的事"
✅ "站起来倒杯水，看看窗外"

❌ "休息一下吧"
✅ "走到窗边，看看最远的地方，深呼吸3次"
```

### 3. 觉察代替批评

```
❌ "你太沉迷了"
✅ "你今天刷的时候，记得什么了吗？"

❌ "你不应该刷这么多"
✅ "你今天2次都是'逃避'，你在逃避什么？"
```

### 4. 陪伴+监督动态平衡

```
0-30分钟：100%陪伴（静默记录）
30-90分钟：60-80%陪伴（温柔提醒）
90分钟+：20-40%陪伴（温和监督）

MVP固定：50%陪伴 + 50%监督
Phase 2：用户可自定义（10%-90%）
```
