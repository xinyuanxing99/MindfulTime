## 🎉 项目正式启动！

**MindfulTime** 是首个基于"觉察-引导-改变"理论的开源时间管理工具。

### ✨ 核心理念

```
❌ 不批评你刷太久
✅ 帮你看见为什么刷

❌ 不冷酷监控
✅ 温暖陪伴 + 温和监督
```

## 🎯 MVP核心功能（设计完成，待开发）

### 1. 📝 记录动机
```
选择平台：抖音/视频号/小红书/B站
输入时长：15/30/60/90分钟
选择动机：
  • 累了想放松 😌
  • 无聊打发时间 😐
  • 逃避工作/学习 😔
  • 主动学习 📖
```

### 2. ⚡️ 能量条机制
```
实时显示能量状态（0-100）：
🌟 精力充沛（90-100）
😊 状态不错（70-89）
😐 还行吧（50-69）
😔 有点累了（30-49）
💤 快撑不住了（0-29）
```

### 3. 🌱 三阶梯渐进式提醒
```
30分钟  → 温柔询问
60分钟  → 友善建议
90分钟  → 温和监督
180分钟 → 必须休息
```

### 4. 📊 "我说了算"占比
```
主动 vs 被动记录
每日占比计算
趋势对比
```

### 5. 🎯 即时引导系统
```
具体行动建议（倒杯水/站起来/看窗外）
能量恢复预测
一键启动微任务
```

### 6. 🌙 晚报系统
```
能量变化曲线
动机分析（今天为什么刷？）
"我说了算"占比
温暖文案生成
```
