# 界面原型设计文档

> **设计工具：** 文字描述 + ASCII示意图（可后续用Figma实现）  
> **设计原则：** 简洁、温暖、直观  
> **核心体验：** 3秒完成记录，一眼看到能量

---

## 🎨 一、设计原则

### **1. 视觉原则**
```
简洁：
├─ 首屏只显示核心信息
├─ 减少视觉干扰
└─ 留白充分

温暖：
├─ 圆角设计
├─ 柔和色彩
├─ 友善文案
└─ Emoji点缀

直观：
├─ 能量条一眼可见
├─ 记录按钮突出
└─ 状态清晰展示
```

### **2. 交互原则**
```
快速：
├─ 记录只需3步
├─ 常用功能一键直达
└─ 无多余跳转

流畅：
├─ 动画60fps
├─ 触觉反馈
└─ 手势操作

容错：
├─ 可撤销
├─ 有确认
└─ 防误触
```

#### 详细设计

**能量卡片：**
```
设计要点：
├─ 大卡片，占屏幕1/3
├─ 能量值大字号显示（48px）
├─ 进度条高度12px，圆角6px
├─ 颜色渐变（绿→黄→橙→红）
└─ Emoji辅助表达情绪

交互：
├─ 点击卡片 → 查看能量变化曲线
└─ 实时动画更新
```

**今日统计：**
```
设计要点：
├─ 列表式展示
├─ 每条带图标（Emoji）
├─ 能量变化用颜色区分（绿色+，红色-）
└─ 轻触查看详情

交互：
├─ 下拉刷新
├─ 点击条目 → 编辑/删除
└─ 长按 → 快捷菜单
```

**主要按钮：**
```
设计要点：
├─ 圆角矩形，高56px
├─ 主按钮"记录刷短视频"用主色
├─ 次按钮"记录做事"用辅色
├─ 按钮有阴影（深度感）
└─ 按下有触觉反馈

交互：
├─ 点击 → 弹出记录弹窗
├─ 长按 → 快捷记录（使用上次选项）
└─ 手势上划 → 打开历史记录
```

#### 详细设计

```
设计要点：
├─ 胶囊形状，圆角24px
├─ 选中：填充色
├─ 未选中：描边
├─ 带Emoji增加辨识度
└─ 可多选（动机可选多个）

交互：
├─ 点击 → 选中/取消
├─ 记住上次选择
└─ 支持自定义选项（Phase 2）
```

**预计影响：**
```
设计要点：
├─ 浅色背景，圆角
├─ 箭头动画（65→50）
├─ 颜色提示（红色表示下降）
└─ 帮用户预判影响

计算逻辑：
├─ 根据时长+动机计算能量变化
├─ 实时更新
└─ 公式：energyChange = calculateEnergyLoss(duration, motivation)
```

```
│                                  │
│  [我去休息] [再刷一会儿]        │ ← 双选择
│                                  │
└─────────────────────────────────┘
```

#### **样式B：温和监督（90-120分钟）**
```
┌─────────────────────────────────┐
│  🌳 刷了1.5小时了                │ ← 图标+文案
├─────────────────────────────────┤
│                                  │

│                                  │

│  1. 站起来，走到窗边             │
│  2. 看看最远的地方（至少20秒）   │
│  3. 深呼吸3次                    │
│                                  │
│  做完了再回来刷也不迟~           │
│                                  │
│  [我去做] [我真的还行]          │
│                                  │
└─────────────────────────────────┘
```

#### **样式C：强监督（180分钟+）**
```
┌─────────────────────────────────┐
│  🚨 刷了3小时了                  │ ← 警示色
├─────────────────────────────────┤
│                                  │
│  你已经刷了3小时了。             │
│                                  │
│  这不是建议，是提醒：            │
│  你需要休息了。                  │
│                                  │
│  现在：                          │
│  1. 站起来                       │
│  2. 离开手机                     │
│  3. 出去走走（至少5分钟）        │
│                                  │
│       [我现在去]                 │ ← 单选择
│                                  │
└─────────────────────────────────┘
```

#### 设计要点

**渐进式设计：**
```
温柔提醒：
├─ 小弹窗（占屏幕1/3）
├─ 圆角大（16px）
├─ 柔和色彩（蓝色/绿色）
└─ 可滑掉

温和监督：
├─ 中等弹窗（占屏幕1/2）
├─ 带图标
├─ 橙色提示
└─ 有遮罩，但可点击背景关闭

强监督：
├─ 全屏弹窗
├─ 红色/警示色
├─ 不可忽略
└─ 背景暗化80%
```

## 📐 五、响应式设计

### **适配策略**

```
小屏幕（<375px）：
├─ 字号适当缩小（-2px）
├─ 间距适当缩小
├─ 单列布局
└─ 隐藏次要信息

中屏幕（375-414px）：
├─ 标准设计
├─ 双列布局（部分）
└─ 完整信息展示

大屏幕（>414px）：
├─ 充分利用空间
├─ 多列布局
├─ 更多信息展示
└─ 考虑平板适配
```
